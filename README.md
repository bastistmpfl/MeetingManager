Minimal local web app to manage coffee and lunch meetups.

## Features
- Calendar and list views
- People + meeting management
- Local storage with IndexedDB
- Export/Import (JSON)

## Quick Start
1) Open `index.html` in your browser
2) Add people, create meetings, export/import as needed

Optional local server:
```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Backup
Use the Export/Import buttons in the header to save/restore data (JSON).
